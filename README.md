# Sistema de Verificação de Idade

Este é um projeto acadêmico desenvolvido em Python que tem como objetivo validar a idade informada pelo usuário e exibir mensagens personalizadas na tela.

# Funcionalidades do Sistema
Identificação: Solicita e armazena o nome do usuário.
Validação: Verifica se a idade digitada é válida (maior ou igual a zero).
Análise de Maioridade: Identifica automaticamente se a pessoa é maior ou menor de idade.
Mensagens Personalizadas: Retorna saudações exclusivas com base nos dados fornecidos.

# Tecnologias Utilizadas
